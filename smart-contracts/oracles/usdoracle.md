# UsdOracle

UsdOracle is a constant price oracle feed that always returns 1 USD in 8 decimal precision.

<br />


## Functions

### `getLatestPrice` 

Returns the constant USD price.

```solidity
    function getLatestPrice()
        view
        returns (
            int256
        );
```
